# AGENTS.md

This file provides guidance to agents when working with code in this repository.

## Deployment Rules
- Deploy to https://bali-island.github.io
- Use GitHub commands in terminal: `git push origin main`
- Tests run via browser: Open https://bali-island.github.io and verify functionality
- Avoid running custom servers locally; use GitHub Pages instead

## Deployment Workflow
1. Commit changes: `git add . && git commit -m "Update AGENTS.md"`
2. Push to GitHub: `git push origin main`
3. Verify deployment at https://bali-island.github.io
4. Test functionality in browser

## Debugging Guidelines
- Use browser DevTools (F12) for CSS debugging
- Check console for errors
- Use `tmp/sysinfo.md
